一、编译前初始化数据库
	1.设置项目：4-Presentation/AuthorityManagement.Web 项目为启动项目(右击该项目即可看到"设置为启动项目")
	2.修改connectionStrings 中AuthorityManagement_Demo的连接sql密码为自己数据库密码
	3.执行vs菜单：工具->nuget程序包管理器->程序包管理器控制台
	4.下vs下方的"程序包管理器控制台"中，设置默认项目为：AuthorityManagement.Data
	5.输入命令：Update-Database -verbose，提示完成则说明数据库初始化成功

二、编译顺序
	右击整个解决方案，编译
	运行程序即可
